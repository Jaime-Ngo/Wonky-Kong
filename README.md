# Wonky-Kong

A retro-inspired platformer — dodge barrels and fireballs, collect garlic for burp attacks, and reach Kong to beat each level.

## Controls

| Action | Keys |
|---|---|
| Move | ← → or A / D |
| Climb | ↑ ↓ or W / S |
| Jump | Spacebar |
| Burp Attack | Tab *(requires garlic)* |

## Run on macOS

1. Install XQuartz: `brew install --cask xquartz`, then log out and back in.
2. Navigate to the game folder: `cd /path/to/WonkyKong`
3. Run: `./WonkyKong`
